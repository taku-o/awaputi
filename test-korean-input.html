🎯 주요 기능

  • 한글 조합 실시간 모니터링 (Composition Events)
  • 자모 분리 현상 자동 수정
  • 스마트 띄어쓰기 (조사 앞 공백 제거)
  • 한글 음절 분석 (초성, 중성, 종성)
  • IME 상태 감지 및 최적화
  • 자동 완성 제안 (데모)
  • 입력 통계 수집

💡 사용 방법

1. "핸들러 연결" 버튼을 클릭하여 한국어 입력 최적화를 활성화합니다.

2. 입력 필드에 한국어를 입력하면서 실시간 상태를 모니터링합니다.

3. 한글 분석 섹션에서 개별 문자의 자모 구성을 확인할 수 있습니다.

4. 스마트 띄어쓰기는 "은", "는", "이", "가" 등의 조사 앞 공백을 자동으로 제거합니다.
